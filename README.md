🌸 MatriNest – Maternal & Child Healthcare Platform
The Safe Nest Every Mom and Baby Deserve

MatriNest is a full-stack healthcare web application developed using Django to provide a centralized platform for maternal and child healthcare services. The system aims to simplify access to healthcare by integrating multiple essential services into a single secure platform.

Designed with a role-based access control (RBAC) system, MatriNest serves three types of users—Patients, Doctors, and Administrators—while ensuring data privacy, secure authentication, and an intuitive user experience.

⚙️ Installation
Clone Repository
git clone https://github.com/your-username/MatriNest.git
cd MatriNest
Create Virtual Environment
python -m venv venv
Activate Environment
Windows

venv\Scripts\activate
Linux / macOS

source venv/bin/activate
Install Dependencies
pip install -r requirements.txt
Apply Migrations
python manage.py migrate
Create Superuser
python manage.py createsuperuser
Run Server
python manage.py runserver
Visit:

http://127.0.0.1:8000/
Admin Panel:

http://127.0.0.1:8000/admin/
