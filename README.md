# 🌸 MatriNest – Maternal & Child Healthcare Platform

> **A Safe Nest for Every Mother and Child**

MatriNest is a **full-stack healthcare web application** built with **Django** that provides a centralized platform for maternal and child healthcare services. It brings together essential healthcare features—including appointment booking, health tracking, emergency support, medicine management, and mental wellness—into a single secure and user-friendly system.

The platform is designed using **Role-Based Access Control (RBAC)** and supports three types of users:

- 👩 Patients
- 👨‍⚕️ Doctors
- 👨‍💼 Administrators

MatriNest focuses on delivering a seamless healthcare experience while ensuring **secure authentication**, **data privacy**, and **efficient healthcare management**.

## ⚙️ Installation

```bash
git clone https://github.com/your-username/MatriNest.git

cd MatriNest

python -m venv venv

# Windows
venv\Scripts\activate

# Linux/macOS
source venv/bin/activate

pip install -r requirements.txt

python manage.py migrate

python manage.py createsuperuser

python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```

Admin Panel:

```
http://127.0.0.1:8000/admin/
```
